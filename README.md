## 简介
本脚手架是一个实验性的项目，用于快速生成一个具备基础配置的完整的single-spa项目，开箱即用，可直接进行业务开发。
## 安装
```
npm install -g icezone-cli
```
## 指令
### icezone-cli init `<name>`

初始化single-spa项目，根据提示对基座应用和子应用进行简单配置
